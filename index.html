Functional machine intelegence

We develop software from user experience to functional programming with Haskel, support and training. Functional programming prevents side-effects and provides referential transparency.

With our platform we are able to built systems that are

  • completely absent of side-effects (financial services, exact calculations, robotic applications etc)
  • able to apply real-time contextual signals and preference filters
  • able to match workflows to percipients
